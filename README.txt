Рисовашка.pyw
Простой графический редактор на Python с использованием библиотеки Tkinter.

Описание
Программа позволяет рисовать на виртуальном холсте, изменять цвета, сохранять рисунки и выполнять другие операции. Интерфейс программы интуитивно понятен и удобен в использовании.

Функции
Рисование: Пользователь может рисовать линии, овалы и прямоугольники различной толщины и цвета.
Изменение цветов: Возможность выбора цвета кисти и фона холста.
Файловые операции: Открытие существующих изображений и сохранение нарисованных рисунков в файлы форматов PNG, BMP, JPG.
Горячие клавиши: Удобные горячие клавиши для быстрого доступа к основным функциям.
Справочная система: Встроенная справка объясняет использование горячих клавиш и основных функций.
Автосохранение: Программа предлагает сохранить изменения перед закрытием окна.
Установка
Для запуска программы необходимо установить следующие библиотеки:


pip install tkinter pillow pyautogui
Запуск
Просто запустите файл Рисовашка.pyw:


Горячие клавиши
F1: Выбор цвета кисти.
F2: Использование ластика.
F3/F4: Уменьшение/увеличение толщины кисти.
F5: Изменение цветовой схемы окна.
F6: Установка цвета фона холста.
F7: Завершение текущего рисунка.
F8: Показ справки.
F9: Переключение формы фигуры (овал/прямоугольник).
F10: Сохранение рисунка.
F11: Открытие файла изображения.
F12: Выбор предустановленной раскраски.
